# Data Science Salaries Analysis

## Description

This project investigates compensation patterns across job roles, experience levels, company sizes, locations, employment types, and remote work settings using data preprocessing, statistical summaries, distribution analysis, correlation analysis, and visualizations to identify key salary trends and workforce insights.

---

## Dataset Overview

- **Dataset:** Data Science Salaries Dataset
- **Total Records:** 151,445
- **Total Features:** 11
- **Time Period:** 2020–2025
- **Domain:** Data Science and Analytics Salaries

### Dataset Attributes

- Work Year
- Experience Level
- Employment Type
- Job Title
- Salary
- Salary Currency
- Salary in USD
- Employee Residence
- Remote Ratio
- Company Location
- Company Size

---

## Objectives

- Analyze salary distributions across the data science industry.
- Examine the relationship between experience level and compensation.
- Compare salaries across job roles and geographic regions.
- Investigate remote work and company size trends.
- Identify key factors influencing salary growth.

## Key Insights

- Experience level is one of the strongest determinants of salary.
- Executive and senior-level positions command significantly higher compensation.
- Salaries show consistent growth across the analyzed period.
- Geographic location significantly influences compensation levels.
- Remote positions often offer competitive salary packages.
- Larger organizations generally provide higher salaries than smaller companies.
- Specialized roles such as Machine Learning Engineer and Research Scientist command higher average salaries than many traditional data roles.
